n 次方差公式

a n - b n = a - b ( ∑ i = 0 n - 1 a n - 1 - i b i )

重要极限补充

lim x → 0 x x = 1
lim x → 0 x ln x = 0
lim n → ∞ 1 + 1 n n = e

等价无穷小

若

α < β α → 0 , β → α

则

α + β → β

完备事件组

刀切全集，不重不漏

概率不等式补充

A B ⊂ A ⊂ A ∪ B P A B ≤ P A ≤ P A + B

互斥

若 AB 互斥，则 A 与 BC 也互斥

所以

P A ∪ B = P A B - ∪ B = P A B - + P B

sin 与 cos 的一些积分

参数为离散值的函数

第一类：函数列

如：

f n x = cos n x n = 1 , 2 , …

为数列

f 1 x = cos x f 2 x = cos 2 x … …

第二类：分段函数

x ≤ 0 , f x = x 1 n ≤ x < 1 n - 1 , f x = 1 n , n = 2 , 3 , …

这是合并的写法， n 取一个值，代表一个分段，不可单纯认为函数分为了两段

贝叶斯与普通条件概率

二者都是为了求

P A i | B

区别在于事件 B 是由什么构成的

贝叶斯

P B = P C A 1 + A 2 + … + A n

普通条件

P B = P A 1 + A 2 + … A n

如果事件 B 是事件组中的某个事件，则用普通条件概率计算

如果事件 B 是指事件组的某个事件与另一个事件同时发生，则用贝叶斯计算

若未知数趋于正无穷，则函数单调有界必 有极限

海涅定理

lim x → x 0 f x = A ⇔ ∀ x n , n → ∞ , x n → x 0 , x n ≠ x 0 , lim n → ∞ f x n = A

积分的放缩

f x ∈ h x , g x ⇒ ∫ a b h x dx ≤ ∫ a b f x dx ≤ ∫ a b g x dx
a ≤ b ≤ c ≤ d , f x ≥ 0 ⇒ ∫ b c f x ≤ ∫ a b f x
f ' x > 0 , ∫ a b f x dx = b - a f ζ , ζ ∈ a , b ⇒ b - a f a ≤ ∫ a b f x dx ≤ b - a f b
